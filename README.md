Each operation can use different server URL defined using `OperationServers` map in the `Configuration`.


## Documentation For Authorization


Authentication schemes defined for the API:
### ApiKeyAuth

- **Type**: API key
- **API key parameter name**: X-Aerostack-Key
- **Location**: HTTP header

Note, each API key must be added to a map of `map[string]APIKey` where the key is: X-Aerostack-Key and passed in as the auth context for each request.

Example

```go
auth := context.WithValue(
		context.Background(),
		aerostack.ContextAPIKeys,
		map[string]aerostack.APIKey{
			"X-Aerostack-Key": {Key: "API_KEY_STRING"},
		},
	)
r, err := client.Service.Operation(auth, args)
```
